# Where are they? (photo tagging app)

This project was made to tie everything I have learned so far together. A photo tagging app
letting users play a "Where's Waldo" type of game.

## Features

- Find 3 characters by clicking on the game image and submitting one of the 3 characters to find.

- Keeps track of your time to add to the leaderboard.
